一、设备固件更新通知
Sony ILME-FX30,ILCE-7M4已推出Ver2.0固件，ILME-FX3推出Ver3.0固件！！
Sony ILCE-7M4(A7M4)
1.增加了对Creators' App应用程序的支持。
*新的Creators' App替代Imaging Edge Mobile应用程序。
*Creators' App可以实现影像从相机到智能手机的传输、遥控拍摄及各种功能。
2. 增加了当使用Creators' App或Imaging Edge Desktop进行遥控拍摄或影像传输时，提高安全性的功能。
*需要将Imaging Edge Desktop更新为最新版本（Ver.3.6或更高版本）。
3.提高了相机的整体稳定性。
ILCE-7M4(A7M4)Ver.2.00固件下载【索尼支持】

Sony ILME-FX3(FX3)
*增加了对Creators' App应用程序的支持。
-新的Creators' App替代Imaging Edge Mobile应用程序。
-Creators' App可以实现影像从摄影机到智能手机的传输、遥控拍摄及各种功能。
*增加了当使用Creators' App或Imaging Edge Desktop进行遥控拍摄或影像传输时，提高安全性的功能。
-需要将Imaging Edge Desktop更新为最新版本（Ver.3.6或更高版本）。
*增加了对呼吸补偿的支持。
*增加了对DCI 4K（4096 x 2160）和24.00p记录的支持。
*录制或回放动态影像的过程中，当显示变形镜头压缩过的影像时，增加了可根据变形镜头的比例校正压缩影像宽比的功能。
*增加了S&Q与动态影像的模式切换，可使用自定义键操作的功能。
*支持文件命名格式“Cam ID + Reel 设置”。
*提高了摄影机的操作稳定性。
